# 🛒 E-commerce "Don Sixto" - Prototipo SPA

Desarrollo de una **Single Page Application (SPA)** interactiva para una tienda online, enfocada en la experiencia de usuario y la eficiencia del código.

### 🚀 Tecnologías Utilizadas
* **JavaScript (Vanilla):** Lógica de navegación, manejo del DOM y carrito de compras.
* **Tailwind CSS:** Diseño responsive y estilizado moderno.
* **HTML5:** Estructura semántica.

### 💡 Soluciones Implementadas
* **Filtrado Dinámico:** Sistema de búsqueda y categorías en tiempo real.
* **Gestión de Estado:** Persistencia del carrito de compras durante la navegación.
* **Diseño Mobile-First:** Optimizado para una navegación fluida en dispositivos móviles.
